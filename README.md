## 🤝 Coding convention
- 기본적인 협업 방식은 github flow를 따릅니다.
- branch명은 `feature/AddReadme` 형태로 작성합니다.
- 커밋 메시지는 `[FEAT] Message` 형태로 작성합니다. (type: feat, fix, chore)
- Pull Request는 한 명 이상의 피어 리뷰를 받고 merge합니다.
